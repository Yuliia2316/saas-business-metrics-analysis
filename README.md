# saas-business-metrics-analysis
SaaS Revenue &amp; Churn Analysis Проєкт з аналізу доходів та утримання користувачів.  SQL (PostgreSQL): розрахунок метрик (MRR, LTV, Churn Rate) за допомогою віконних функцій.  Tableau: візуалізація динаміки виручки, сегментація доходу та когортний аналіз. Результат: дашборд для моніторингу ефективності підписної моделі.
